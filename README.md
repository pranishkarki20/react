# AI E-Commerce Platform

An AI-powered e-commerce project built for both customers and sellers. The goal of this platform is to create a smarter shopping experience for users while also giving sellers intelligent tools to manage products, understand performance, and grow their business.

## Project Vision

This project combines a modern e-commerce interface with AI-driven support features. It is designed to help:

- users discover products more easily
- sellers manage their store more efficiently
- both sides benefit from intelligent automation and insights

## Core Features

### User Features

- Browse products by category
- View product listings in a clean interface
- Contact page for customer communication
- Authentication flow for login and signup
- Smooth navigation with React Router
- Responsive UI with animations

### Seller Features

- Seller/admin dashboard
- Product management support
- AI-based suggestions for sellers
- Automatic report generation and printing support
- Insights to help sellers improve product decisions
- Smarter store management workflow

## AI Feature Ideas

This project is being built around practical AI assistance for e-commerce. Some planned or ongoing ideas include:

- automatic sales report generation
- printable reports for seller review
- smart product suggestions for sellers
- recommendations based on product activity and trends
- decision support for inventory and sales improvement
- personalized product suggestions for users in future versions

## Getting Started

### Install dependencies

```bash
npm install
```

### Run frontend

```bash
npm run dev
```

### Run backend

```bash
npm run server
```
